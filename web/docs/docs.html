Getting Started

System Requirements

  • Java Development Kit (JDK) 17 or higher
  • Gradle (bundled with our distribution)
  • Basic familiarity with command-line interfaces

Quick Start

  1. Download and Install:
    git clone https://github.com/VerisimilitudeX/DNAnalyzer.git
  2. Navigate to Directory:
    cd DNAnalyzer
  3. Build the Project:
    ./gradlew clean bootJar

Features in Detail

Start and Stop Codon Detection

Understanding where a protein begins and ends is crucial for interpreting genetic data. DNAnalyzer pinpoints these codons, providing insights into the protein synthesis process and potential areas of interest for further study.

High Coverage Region Identification

Regions rich in guanine and cytosine (GC-content) often indicate important promoter sequences. Our tool analyzes these areas, helping to predict gene activity and regulatory potential.

Neurodevelopmental Disorders Analysis

Longer genes are sometimes linked to disorders affecting brain development. By detecting genetic markers associated with conditions like autism and ADHD, DNAnalyzer supports research into complex neurodevelopmental issues.

Core Promoter Element Analysis

Essential sequences such as BRE, TATA, INR, and DPE are critical for initiating transcription. DNAnalyzer not only identifies these elements but also interprets their roles in gene regulation.

CLI Usage

Basic Commands

java -jar build/libs/DNAnalyzer.jar [file] [options]

Common Options

  • --amino=X - Specify start amino acid (e.g., ser, met)
  • --min=X - Minimum sequence length
  • --max=X - Maximum sequence length
  • -r - Reverse sequence analysis

Example Usage

java -jar build/libs/DNAnalyzer.jar assets/dna/random/dnalong.fa --amino=ser --min=16450 --max=520218 -r

API Reference

Core Classes

  • DNAAnalysis

    Main analysis class containing core functionality

  • DNAMutation

    Handles sequence mutations and variations

  • DNADataUploader

    Manages file uploads and data preprocessing
