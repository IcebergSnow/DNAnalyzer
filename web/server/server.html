Connection Instructions

To start the DNAnalyzer server, follow these steps:

  1. Make gradlew executable:

    chmod +x ./gradlew

  2. Clean build (this may take a minute):

    ./gradlew clean bootJar

  3. Start the server:

    java -jar build/libs/DNAnalyzer.jar
The server will start on localhost:8081. Make sure you have Java 17 or higher installed.
